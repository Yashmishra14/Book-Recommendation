About Us

Welcome to the Book Recommendation App! Our mission is to help book lovers find the best books based on their preferences and ratings from other readers. Whether you're into fiction, non-fiction, fantasy, or biographies, our app is here to make your next book selection easier and more enjoyable.

Technology Stack

We built this application using modern web technologies to ensure smooth performance and an enjoyable user experience.

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python (Flask/Django), REST APIs
  • Database: MySql
  • Data Processing: Pandas, Scikit-Learn (for recommendations)
  • Deployment: Heroku/AWS
